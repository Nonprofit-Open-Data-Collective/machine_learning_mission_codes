## Installing Packages

```r
install.packages( c( "quanteda", "purrr", "xml2", "dplyr", "XML", "jsonlite" ) )
devtools::install_github( 'ultinomics/xmltools' )
library( xmltools )
library( purrr )
library( xml2 )
library( dplyr )
library( XML )
library( jsonlite )
```
